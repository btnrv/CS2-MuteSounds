# MuteSounds

A CounterStrikeSharp plugin for CS2 that allows muting various game sounds.

## Installation

1. Install [CS2MenuManager](https://github.com/schwarper/CS2MenuManager)
2. Download the latest release, drop files into *addons/counterstrikesharp/plugins/*
3. Restart your CS2 server.
4. Configure the plugin from *addons/counterstrikesharp/configs/plugins/MuteSounds*

## Features

- Mute footsteps
- Mute knife sounds
- Mute gun sounds
- Mute player hurt sounds
- Mute MVP music
- Target-based filtering (Everyone/Teammates/Enemies)
- Menu based controls

*Supported languages: TR, EN*
